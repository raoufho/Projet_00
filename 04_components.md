## 1.5 المكونات الإلكترونية الأساسية

الآن بعد أن فهمت المبادئ الأساسية، دعنا ننظر إلى **اللاعبين الرئيسيين** في الإلكترونيات.

---

## 1. الثنائي (Diode)

### 🔬 التعريف:
مكون إلكتروني بسيط يسمح للتيار بالمرور في **اتجاه واحد فقط**.

### 🎓 الشرح:
تخيل أنه مثل **صمام الماء**:
- **يسمح بسريان الماء في اتجاه واحد** (من طرف للآخر)
- **يمنع العودة الخلفية** (لا يسري معكوس)

### 📊 الرمز والخصائص:

```
الرمز الكهربائي:
      │      (الطرف الموجب - Anode)
      △
    ◁───  الثنائي
     │
     │      (الطرف السالب - Cathode)

التوصيل الصحيح (Forward Biased):
(+) ─────△────── (−)
     يسري التيار ✓

التوصيل المعكوس (Reverse Biased):
(−) ─────△────── (+)
     لا يسري التيار ✗ (مقاومة كبيرة جداً)
```

### 🚗 تطبيقات في السيارات:

| التطبيق | الوصف |
|---------|-------|
| **مصحح الدينامو** | يحول AC إلى DC |
| **حماية الدوائس** | منع التيار العكسي |
| **الإضاءة العكسية** | بعض المصابيح الحديثة |
| **وحدات التحكم** | حماية المدخلات |

### 📌 الأنواع الشائعة:

```
1. Standard Diode (1N4007)
   - استخدام عام
   - تحمل 1A تقريباً
   - موجود في معظم الدوائس

2. Schottky Diode
   - سريع جداً
   - فقدان جهد أقل
   - استخدام في مصححات عالية الكفاءة

3. Zener Diode
   - خاص لتثبيت الجهد
   - سيتم شرحه مع المنظمات
```

## 2. الترانزستور (Transistor)

### 🔬 التعريف:
مكون إلكتروني يسمح بـ **التحكم في تيار كبير باستخدام تيار صغير**.

### 🎓 الشرح:
تخيله مثل **مفتاح مائي يتحكم فيه محرك كهربائي صغير**:
- **المحرك الصغير** = التيار الصغير (إشارة التحكم)
- **الماء الكبير** = التيار الكبير (التيار المراد التحكم به)

### 📊 الرمز والأنواع:

```
ترانزستور NPN (الأكثر شيوعاً في السيارات):
      
      C (Collector - المجمع)
      │
      ├─────┐
      │     │ ─── B (Base - القاعدة)
      ├─────┤
      │     │
      E (Emitter - الباعث)

العمل:
- تيار صغير في B (Base) يسبب تيار كبير من C إلى E
- يعمل كـ "مفتاح إلكتروني"
```

### 🚗 تطبيقات في السيارات:

| التطبيق | الوصف |
|---------|-------|
| **مفاتيح إلكترونية** | التحكم في المحركات والمصابيح |
| **مضخمات الإشارة** | تقوية إشارات الحساسات الضعيفة |
| **محركات الريليه والكويل** | التحكم في الأحمال الكبيرة |
| **وحدات التحكم** | جزء من كل جهاز إلكتروني |

### 📌 أمثلة على الأداء:

```
مثال: التحكم في مصباح 50A عبر إشارة 0.1A

┌─────────────────────┐
│  وحدة التحكم      │
│  (ECU)            │
│  إخراج 0.1A      ├─────┐
└─────────────────────┘     │
                            │ Base
                            ├──┐
                        ┌───┤  │ Transistor
                        │   └──┤
                        │      ├─── Collector
                        │      │    ↓ 50A
                        │      │
                    +12V│     ─┴─
                        │      │
                        │     ╱ │ مصباح 50A
                        │    │  │
                        │     ╲ │
                        │      ─┬─
                        │      │
                       GND    GND

النتيجة:
- 0.1A من ECU يتحكم في 50A للمصباح!
- هذا هو "السحر" الإلكتروني!
```

## 3. الثنائي الباعث للضوء (LED - Light Emitting Diode)

### 🔬 التعريف:
نوع خاص من الثنائيات يُصدر **ضوءاً** عند مرور التيار من خلاله.

### 🎓 الخصائص:
- ✓ استهلاك طاقة منخفض جداً
- ✓ عمر طويل (50,000+ ساعة)
- ✓ ألوان متعددة (أحمر، أبيض، أزرق، إلخ)
- ✓ تشغيل سريع (بدون تأخير)
- ✗ حساس للقطبية (يجب توصيله صحيح)

### 📊 الرمز والخصائص:

```
رمز LED:
      │ Anode (+)
      △
    ◁┃  LED (الخطوط تمثل الضوء)
    ┃│
      │ Cathode (−)

أطول من:  الطرف الموجب
أقصر من:  الطرف السالب

جهد التشغيل النموذجي:
  • أحمر: 1.8 − 2.2V
  • أخضر: 2.0 − 2.5V
  • أزرق: 3.0 − 3.5V
  • أبيض: 3.0 − 3.5V
```

### 🚗 تطبيقات في السيارات:

| التطبيق | الفائدة |
|---------|--------|
| **مؤشرات اللوحة** | استهلاك أقل = بطارية تدوم أطول |
| **مصابيح الإضاءة الأمامية (LED)** | أقوى وأطول عمراً من الهالوجين |
| **مصابيح الإضاءة الخلفية** | أسرع استجابة (أمان أفضل) |
| **شاشات المعلومات** | دقة لونية عالية |
| **أضواء التحذير** | تمايز واضح |

### 🔴 تحذير مهم:

```
❌ خطأ شائع: توصيل LED مباشرة على 12V

┌─────┬─────┐
│  +  │  −  │
│12V  │GND  │
└─────┬─────┘
      │
      ▼
    ─┬─
     │ LED ← سيحترق في الحال!
    ─┴─
      │
     GND

السبب: 12V كبير جداً للـ LED (يحتاج 2V فقط)
الحل: استخدام "مقاومة محدودة للتيار" (Series Resistor)
```

## 4. المقاومة (Resistor)

### 🔬 التعريف:
مكون يُعاكس سريان التيار بقيمة محددة.

### 📊 الرمز والخصائص:

```
الرمز:
   ───────
   │ ≈≈≈ │  (الخطوط تمثل المقاومة)
   ───────

التسلسل اللوني (كود الألوان):
أول لون:    الرقم الأول
ثاني لون:   الرقم الثاني
ثالث لون:   عدد الأصفار
رابع لون:   دقة التحمل

مثال: أحمر - بنفسجي - أحمر - ذهبي
      2    7        2        5%
      = 27 × 100 = 2700 Ω = 2.7 kΩ (±5%)
```

### 🚗 تطبيقات في السيارات:

| التطبيق | الوصف |
|---------|-------|
| **محدود التيار** | حماية الثنائيات والـ LED |
| **مقسمات الجهد** | قياس الحساسات |
| **دوائس التصفية** | تنظيف الإشارات |
| **تحديد مستويات العمل** | ضبط نقاط التشغيل |

## 5. المكثف (Capacitor)

### 🔬 التعريف:
مكون يخزّن **الطاقة الكهربائية** ويطلقها.

### 🎓 الشرح:
تخيله مثل **حمام سباحة مصغر**:
- **شحنه** = ملء حوض السباحة
- **تفريغه** = فتح الماء

### 📊 الخصائص:

```
الرمز:
   ┌─┬─┐
───┤ │ ├───  capacitor
   └─┴─┘

الوحدة: Farad (F)
الأحجام الشائعة:
  • microfarad (µF) = 10⁻⁶ F
  • millifarad (mF) = 10⁻³ F
  • nanofarad (nF) = 10⁻⁹ F
```

### 🚗 تطبيقات في السيارات:

| التطبيق | الوصف |
|---------|-------|
| **تنظيف إشارات الحساسات** | إزالة الضوضاء |
| **حماية الدوائس** | امتصاص القمم الكهربائية |
| **مصابيح الفلاش** | تخزين الطاقة |
| **وحدات التحكم** | تثبيت الجهد |

## 🔍 مقارنة المكونات:

| المكون | الوظيفة | الاستخدام | التطبيق |
|--------|---------|-----------|---------|
| **Diode** | مفتاح أحادي الاتجاه | تصحيح AC | الدينامو |
| **Transistor** | مفتاح إلكتروني | تضخيم | وحدات التحكم |
| **LED** | مصدر ضوء | إضاءة | المؤشرات |
| **Resistor** | معاكس التيار | حماية | الدوائس |
| **Capacitor** | خزان طاقة | تصفية | تثبيت الجهد |

## ⚠️ ملاحظات أمان مهمة:

```
عند التعامل مع المكونات:

✓ افحص القطبية قبل التوصيل
✓ لا توصل أجزاء قيمة عشوائياً
✓ استخدم مقاومات محدودة التيار عند الحاجة
✓ لا تزيد الجهد أو التيار عن الحد الأقصى
✓ راقب الحرارة (إذا سخن = مشكلة)
```
