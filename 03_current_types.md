# ⚡ الملف الثالث: أنواع التيار الكهربائي
## Electrical Current Types: DC vs AC

---

## 1.4 أنواع التيار الكهربائي

في السيارات، ستتعامل مع نوعين أساسيين من التيار. فهم الفرق بينهما حاسم جداً.

---

## النوع الأول: التيار المستمر (Direct Current - DC)

### 🔬 التعريف:
تيار كهربائي يسري في **اتجاه واحد ثابت** لا يتغير مع الزمن.

### 📊 رسم توضيحي (ASCII):

```
الجهد عبر الزمن:
     V
     |
   12V ├─────────────────────────────
     |                               
     |                               
   0V ├─────────────────────────────
     └───────────────────────→ الزمن
     
الخصائص:
✓ ثابت (لا يتغير مع الزمن)
✓ اتجاه واحد (من + إلى −)
✓ قابل للتخزين في البطاريات
```

### 📐 المعادلة:
```
V(t) = V_constant = 12V (في جميع الأوقات)
I(t) = I_constant (إذا ظلت المقاومة ثابتة)
```

### 🚗 تطبيقات في السيارات:

| النظام | الوصف | مثال |
|--------|-------|------|
| **البطارية** | مصدر DC أساسي 12V | جهد ثابت دائماً |
| **الدينامو (بعد التصحيح)** | ينتج AC لكن يتم تصحيحه إلى DC | 13.5-14.5V |
| **محرك البدء** | يعمل على DC 12V | يشغل المحرك |
| **أنظمة التحكم الإلكترونية** | جميعها تستخدم DC | المعالجات والدوائس |
| **الإضاءة** | تعمل على DC (مع بعض المصابيح الحديثة) | مصابيح LED وHalogen |

### 📌 مميزات DC:
- ✓ **بسيط وآمن نسبياً** - لا يسبب حروق من الاتصال البسيط
- ✓ **سهل التخزين** - يمكن تخزينه في البطاريات
- ✓ **يناسب المحركات والحساسات** - معظم الأجهزة تستخدمه
- ✓ **قابل للتحكم** - سهل الحد من التيار أو القوة
- ✗ **يفقد الطاقة أثناء النقل** - انخفاض جهد في الأسلاك الطويلة

### 🔍 مثال عملي - دائرة بسيطة:

```
البطارية (DC 12V)
    │
    ├────────┐
    │     (R) مقاومة 2.4Ω
    │        │
    ├────────┤
    │        │
    │    ┌───┴────┐
    │    │ LED    │
    │    └────────┘
    │        │
   GND─────┴────────

الحساب:
  V = 12V (DC ثابت)
  R = 2.4Ω (ثابت)
  I = V / R = 12V / 2.4Ω = 5A (ثابت)
  P = V × I = 12V × 5A = 60W (ثابت)
```

## النوع الثاني: التيار المتردد (Alternating Current - AC)

### 🔬 التعريف:
تيار كهربائي يتغير اتجاهه وقيمته **بشكل دوري** مع الزمن (عادة بشكل جيبي).

### 📊 رسم توضيحي (ASCII):

```
الجهد عبر الزمن (تيار متردد 50 Hz):
      V
      |     ╱╲      ╱╲      ╱╲
      |   ╱    ╲  ╱    ╲  ╱    ╲
  +14V┤ ╱        ╲╱      ╲╱      
      |╱                          
    0V├───────────────────────────
      |╲                          
  −14V┤ ╲        ╱╲      ╱╲      ╱
      |   ╲    ╱  ╲    ╱  ╲    ╱
      |     ╲╱      ╲╱      ╲╱
      └──────────────────→ الزمن
      
الخصائص:
✓ يتغير مع الزمن (دوري)
✓ اتجاهان (موجب وسالب)
✓ يحتاج تصحيح لاستخدامه في السيارات
```

### 📐 المعادلة:
```
V(t) = V_max × sin(2πft)

حيث:
  V_max = أقصى جهد (Peak Voltage)
  f = التردد (Frequency) بـ Hz
  t = الزمن بـ ثانية
  
مثال:
  V(t) = 14V × sin(2π × 50Hz × t)
  
هذا يعني الجهد يتغير من −14V إلى +14V
وتكرار 50 مرة في الثانية
```

### 🎯 المصطلحات المهمة:

| المصطلح | التعريف | المثال |
|--------|---------|--------|
| **V_max (Peak)** | أقصى قيمة | 14V |
| **V_rms (RMS)** | القيمة الفعالة | V_max / √2 = 9.9V |
| **التردد (f)** | عدد التكرارات في الثانية | 50 Hz أو 60 Hz |
| **الدورة (Period)** | الزمن لدورة واحدة | 1/f = 20 ms |

### 🚗 تطبيقات في السيارات:

| النظام | الوصف | ملاحظة |
|--------|-------|---------|
| **الدينامو (قبل التصحيح)** | ينتج AC 3 طور | يتم تصحيحه لاحقاً |
| **أجهزة الاختبار (Oscilloscope)** | تعرض AC في الإشارات | قراءة الموجات |
| **الإشارات من الحساسات** | قد تكون AC (مثل حساس السرعة) | تحتاج معالجة |
| **الشبكات الكهربائية المنزلية** | AC 220V (ليس في السيارات!) | مثال للمقارنة |

### 📌 مميزات AC:
- ✓ **قابل للنقل لمسافات طويلة** - بدون فقدان كبير
- ✓ **سهل التحويل من فولتاج لآخر** - باستخدام محولات
- ✓ **كفاءة عالية في النقل** - أقل فقدان حرارة
- ✗ **معقد نسبياً** - يحتاج معدات تصحيح
- ✗ **صعب التخزين** - لا يمكن تخزينه في البطاريات مباشرة
- ✗ **قد يكون خطيراً** - التيار المتردد أكثر خطورة من DC

## 🔄 التحويل من AC إلى DC

في السيارات الحديثة، الدينامو ينتج **AC**، لكن البطارية والأجهزة تحتاج **DC**.

### المشكلة والحل:

```
الدينامو (AC 3-طور)
    │
    ▼
❌ المشكلة: البطارية تحتاج DC فقط!

الحل:
    │
    ▼
┌─────────────────────┐
│ Rectifier (Diodes)  │  يحول AC → DC
│ (مصحح)              │
└─────────────────────┘
    │
    ▼
┌─────────────────────┐
│ Regulator (ICs)     │  يثبت الجهد
│ (منظم)              │  على 13.5 − 14.5V
└─────────────────────┘
    │
    ▼
  البطارية والأجهزة
  (DC نظيف تماماً) ✓
```

### 📐 المعادلة:

```
Rectifier (المصحح - يستخدم ديودات):
Input:  V_AC(t) = 14V × sin(2πft)
Output: V_DC = |14V × sin(2πft)| (قيمة مطلقة)

النتيجة: تيار يسري في اتجاه واحد فقط

Regulator (المنظم - يستخدم ICs):
Input:  V_DC متغير (من Rectifier)
Output: V_DC = ثابت 13.5V
```

### 🔍 رسم توضيحي للعملية:

```
المرحلة الأولى: AC من الدينامو
   +14V ╱╲      ╱╲
       ╱  ╲    ╱  ╲
    0V╱────╲──╱────╲──
      ╲      ╲╱      
       ╲    ╱╲      
   −14V ╲╱    ╲

المرحلة الثانية: بعد المصحح (Rectifier)
   +14V ┌──┐  ┌──┐
        │  │  │  │
    0V ─┴──┴──┴──┴──
        │
        │ (موجي قليلاً)

المرحلة الثالثة: بعد المنظم (Regulator)
   +14V ─────────────
        │
   13.5V├─────────────
        │
    0V  ├─────────────
        │
```

## 🆚 المقارنة بين DC و AC

### جدول المقارنة الشامل:

| الخاصية | DC | AC |
|--------|----|----|
| **الاتجاه** | واحد فقط | متناوب |
| **الثبات** | ثابت | متغير |
| **التخزين** | يُخزن بسهولة | صعب التخزين |
| **الأمان** | أكثر أماناً | أكثر خطورة |
| **النقل** | فقدان في الأسلاك الطويلة | كفاءة عالية |
| **التحويل** | صعب التحويل | سهل التحويل |
| **الاستخدام في السيارات** | الأساسي | في الدينامو قبل التصحيح |
| **التطبيقات** | البطاريات، الأجهزة | الشبكة الكهربائية |

### مثال عملي:

```
نفس الحمل (مصباح 60W) على:

DC 12V:
  I = P / V = 60W / 12V = 5A
  ← تيار ثابت ثابت

AC 12V (RMS):
  I = P / V = 60W / 12V = 5A (RMS)
  ← التيار يتغير من +7A إلى −7A
  ← أكثر تعقيداً!

النتيجة:
  DC أبسط وأكثر أماناً للسيارات
```

## ⚡ الخصائص المهمة للفنيين

### عند التعامل مع DC:
```
✓ يمكن قياسه بـ Multimeter مباشرة
✓ آمن نسبياً للمس (إذا كان الجهد < 50V)
✓ يسري من + إلى − دائماً
✓ لا يحتاج إلى قلق من "الاتجاه" في الأجهزة
```

### عند التعامل مع AC:
```
✓ يحتاج Oscilloscope لرؤية الموجة
✗ أكثر خطورة من DC (نفس القاعدة 50V)
✗ الاتجاه يتغير باستمرار
✓ يتم نقله بكفاءة في المسافات الطويلة
```
