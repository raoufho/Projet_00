## Core Concepts: Charge, Current, Voltage, Resistance

## 1.3 المفاهيم الأساسية الأربعة

كل شيء في الكهرباء يدور حول أربع كميات أساسية. تعلم هذه الأربع وستفهم **80%** مما تحتاجه:

---

## 1️⃣ الشحنة الكهربائية (Electrical Charge)

### 🔬 التعريف:
الشحنة الكهربائية هي خاصية فيزيائية للمادة تجعلها تتفاعل مع المجالات الكهربائية والمغناطيسية.

### 📐 الرمز والوحدة:
- **الرمز:** Q أو q
- **الوحدة:** Coulomb (كولوم) = C
- **الرمز الاختصار:** 1 C = الشحنة التي ينقلها تيار 1 أمبير في 1 ثانية

### 🎓 الشرح المبسط:
تخيل أن الذرات مثل الكرات المغناطيسية:
- بعضها له **نقص في الإلكترونات** ويميل للجذب (شحنة موجبة: +)
- بعضها له **زيادة في الإلكترونات** ويميل للدفع (شحنة سالبة: -)

عندما تجبر الإلكترونات على التحرك من مكان إلى آخر، تحدث **الشحنة الكهربائية**.

### 🚗 مثال من السيارات:
في البطارية، عندما تحدث تفاعلات كيميائية، يتم فصل الإلكترونات من الذرات:
- **الطرف السالب (−):** تتراكم الإلكترونات الزائدة
- **الطرف الموجب (+):** ينقص الإلكترونات
- **الفرق المحصل:** فارق الجهد 12V (في بطارية السيارة)

### 📊 جدول الشحنات:

| المكون | الشحنة | الوصف |
|-------|--------|-------|
| **الإلكترون** | −1.6 × 10⁻¹⁹ C | وحدة الشحنة السالبة |
| **البروتون** | +1.6 × 10⁻¹⁹ C | وحدة الشحنة الموجبة |
| **بطارية السيارة (طرف −)** | ~10,000 C | شحنة سالبة محترمة |
| **بطارية السيارة (طرف +)** | ~10,000 C | شحنة موجبة محترمة |

## 2️⃣ التيار الكهربائي (Electrical Current)

### 🔬 التعريف:
التيار الكهربائي هو **حركة الشحنات الكهربائية** عبر موصل.

### 📐 الرمز والوحدة:
- **الرمز:** I (من كلمة Intensity)
- **الوحدة:** Ampere (أمبير) = A
- **الصيغة:**
```
I = Q / t

حيث:
  I = التيار (A)
  Q = الشحنة (C)
  t = الزمن (s)
```

### 🎓 الشرح المبسط:
تخيل أن السلك مثل أنبوبة ماء:
- **الماء المتدفق** = الإلكترونات المتحركة = **التيار الكهربائي**
- **كمية الماء في الثانية** = عدد الإلكترونات في الثانية = **قيمة التيار**

**تيار 1 أمبير** = مرور **6.24 × 10¹⁸ إلكترون** في كل ثانية

### 🚗 أمثلة من السيارات:

| النظام | التيار النموذجي | الملاحظة |
|--------|----------------|----------|
| **مصابيح الإضاءة الداخلية** | 0.2 − 0.5 A | تيار صغير |
| **مصابيح الأمامية (LED)** | 2 − 5 A | متوسط |
| **مصابيح الأمامية (Xenon)** | 6 − 8 A | عالي |
| **محرك الفرش الكهربائي** | 10 − 20 A | عالي جداً |
| **موتور البدء** | 200 − 400 A | عالي جداً جداً! |
| **المحرك الكهربائي (EV)** | 100 − 300 A | أقوى استخدام |

### ⚠️ تحذير مهم:
التيار العالي **خطير جداً**! حتى 50 mA (ميلي أمبير) قد تكون كافية لإيقاف نبض القلب.

## 3️⃣ الجهد الكهربائي (Electrical Voltage)

### 🔬 التعريف:
الجهد الكهربائي هو **الفرق في الطاقة بين نقطتين** في دائرة كهربائية.

### 📐 الرمز والوحدة:
- **الرمز:** V أو U
- **الوحدة:** Volt (فولت) = V
- **التعريف الرياضي:**
```
V = W / Q

حيث:
  V = الجهد (V)
  W = الطاقة (J - جول)
  Q = الشحنة (C)
```

### 🎓 الشرح المبسط (مقارنة بالماء):
تخيل خزان ماء على ارتفاع مختلف:
- **الارتفاع بين نقطتين** = فرق الجهد الكهربائي
- **كلما زاد الارتفاع** = كلما زاد الضغط = كلما زاد الجهد
- **الماء يتدفق من الأعلى للأسفل** = الإلكترونات تتدفق من − إلى +

### 🚗 أمثلة من السيارات:

```
البطارية (12V)
    |
    +─── مصباح الرأس ──────────┐
    |                         |
    +─── محرك الفرش ──────────┤  كل واحد يأخذ 12V
    |                         |
    +─── نظام التحكم ─────────┘

التيار واحد، لكن الجهد موزع!
```

| المكون | الجهد النموذجي |
|--------|--------------|
| **البطارية (الحالة الطبيعية)** | 12.6 − 13.8 V |
| **البطارية (محرك شغال)** | 13.5 − 14.5 V |
| **البطارية (محرك متوقف)** | 11.5 − 12.0 V |
| **نظام الشحن (الدينامو)** | 13.5 − 15.0 V |
| **سيارة كهربائية (بطارية عالية الجهد)** | 300 − 400 V |

### 🔴 نقطة حرجة:
في السيارات الهجينة والكهربائية، قد يصل الجهد إلى **400 فولت**!
- هذا الجهد **قاتل** إذا لمسته مباشرة
- يجب اتخاذ إجراءات أمان صارمة جداً (سيتم شرحها لاحقاً)

## 4️⃣ المقاومة الكهربائية (Electrical Resistance)

### 🔬 التعريف:
المقاومة الكهربائية هي **مقدار معارضة المادة لسريان التيار**.

### 📐 الرمز والوحدة:
- **الرمز:** R
- **الوحدة:** Ohm (أوم) = Ω
- **الصيغة:**
```
R = V / I

حيث:
  R = المقاومة (Ω)
  V = الجهد (V)
  I = التيار (A)
```

### 🎓 الشرح المبسط:
تخيل أنابيب الماء:
- **أنبوبة واسعة** = مقاومة منخفضة = يسري الماء بسهولة
- **أنبوبة ضيقة/مسدودة** = مقاومة عالية = يصعب سريان الماء
- **أنبوبة مغلقة تماماً** = مقاومة لا نهائية = لا يسري ماء

### 🚗 أمثلة من السيارات:

| المكون | المقاومة | النوع |
|--------|---------|--------|
| **سلك نحاسي سميك** | 0.001 − 0.01 Ω | منخفضة جداً |
| **مصباح الإضاءة** | 50 − 100 Ω | منخفضة |
| **محرك كهربائي شغال** | 0.1 − 1 Ω | منخفضة جداً |
| **حساس درجة حرارة** | 1,000 − 100,000 Ω | عالية |
| **سلك مقطوع** | ∞ (لا نهائية) | مفتوح |
| **تلامس ضعيف** | 0.1 − 10 Ω (قد يبدو طبيعياً!) | **خطر خفي** |

### 🔍 دراسة حالة:

**الحالة:** سيارة تواجه مشكلة في إضاءة المصباح الأمامي

**القياسات:**
- جهد البطارية: 12V ✓
- جهد عند المصباح: 11.2V (نقص 0.8V)
- التيار: 5A

**التحليل:**
```
الفارق = 0.8V
التيار = 5A

المقاومة الزائدة = 0.8V / 5A = 0.16 Ω

هذه مقاومة صغيرة، لكنها تسبب:
- نقص في السطوع (السلك ينقل 5 أمبير × 0.16 Ω = 0.8W حرارة!)
- قد تكون التلامسات رطبة أو الكابل يحتاج استبدال
```

## 🔗 العلاقة بين المفاهيم الأربعة

### 💡 قانون أوم (Ohm's Law):

الربط بين الثلاثة (V, I, R):

```
V = I × R

أو:

I = V / R

أو:

R = V / I

هذا يعني:
- إذا زاد الجهد → يزيد التيار (إذا ظلت المقاومة ثابتة)
- إذا زادت المقاومة → يقل التيار (إذا ظل الجهد ثابتاً)
- إذا قل التيار → قل الجهد (إذا ظلت المقاومة ثابتة)
```

### 📊 رسم توضيحي (ASCII):

```
العلاقة الرياضية:

    V (الجهد)
    ↓
┌─────────────────┐
│   V = I × R     │
│   V / I = R     │
│   V / R = I     │
└─────────────────┘
    ↑           ↑
    I (التيار)  R (المقاومة)

مثال عملي:
  البطارية: 12V
  مصباح: 60W
  
  التيار: I = P / V = 60W / 12V = 5A
  المقاومة: R = V / I = 12V / 5A = 2.4 Ω
```

### 🚗 تطبيق عملي:

إذا أردت زيادة سطوع مصباح:
```
الطريقة 1 (خطأ):
  زيادة الجهد من 12V إلى 14V
  ← يزيد التيار ← المصباح يسخن ← يحترق!

الطريقة الصحيحة:
  التأكد من توصيل صحيح (مقاومة منخفضة)
  ← يسري تيار أكبر بنفس الجهد
  ← المصباح يضيء أقوى بأمان!
```

## 4️⃣-bis القوة الكهربائية (Electrical Power)

### 🔬 التعريف:
القوة (أو الاستطاعة) هي **معدل استهلاك الطاقة**.

### 📐 الرمز والوحدة:
- **الرمز:** P
- **الوحدة:** Watt (واط) = W
- **الصيغة الأساسية:**
```
P = V × I

حيث:
  P = القوة (W)
  V = الجهد (V)
  I = التيار (A)
```

### 📐 صيغ بديلة:
```
باستخدام قانون أوم:

P = V × I = V × (V/R) = V²/R
P = V × I = (I×R) × I = I² × R
```

### 🚗 أمثلة من السيارات:

| الجهاز | الجهد | التيار | القوة |
|--------|------|--------|--------|
| **مصباح داخلي** | 12V | 0.3A | 3.6W |
| **مصباح أمامي (قياسي)** | 12V | 5A | 60W |
| **مصباح أمامي (Xenon)** | 12V | 6.7A | 80W |
| **محرك فرشاة** | 12V | 15A | 180W |
| **موتور البدء** | 12V | 300A | 3,600W |

### 💡 الفهم العملي:

```
قوة أعلى = استهلاك أكثر:

مصباح LED (4W):
  - استهلاك قليل
  - حرارة قليلة
  - عمر أطول

مصباح Xenon (80W):
  - استهلاك أكثر
  - حرارة أكثر
  - يحتاج توصيل أقوى
  - يستنزف البطارية أسرع
```

## 📊 ملخص المفاهيم الأربعة

```
┌──────────────────────────────────────────────┐
│  المفاهيم الأربعة الأساسية                  │
├──────────────────────────────────────────────┤
│                                              │
│  Q (Charge/الشحنة)                          │
│  └─ الكمية الكلية للإلكترونات              │
│                                              │
│  I (Current/التيار)                         │
│  └─ حركة الشحنات: I = Q / t                │
│                                              │
│  V (Voltage/الجهد)                          │
│  └─ فرق الطاقة: V = W / Q                  │
│                                              │
│  R (Resistance/المقاومة)                    │
│  └─ معارضة التدفق: R = V / I              │
│                                              │
│  P (Power/القوة)                            │
│  └─ استهلاك الطاقة: P = V × I             │
│                                              │
│  العلاقات الأساسية:                         │
│  • V = I × R (قانون أوم)                   │
│  • P = V × I (القوة)                       │
│  • P = I² × R = V²/R (صيغ بديلة)          │
│                                              │
└──────────────────────────────────────────────┘
```
